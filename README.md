## Description

Nestjs test backend application for tracking pizza orders.

Setup to run locally with Docker.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod

```
